# DU_WIKI
대구대학교 학생들을 위한 위키 페이지
>[DU_WIKI](https://duwiki.100won.duckdns.org/index.php/%EB%8C%80%EB%AC%B8)
## 프로젝트 목표
대구대학교 학생들에게 유익한 정보를 공유하고, 정보의 장을 만들어주기 위함.
